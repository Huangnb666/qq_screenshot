# qq_screenshot

一个基于 **PyQt6** 打造的轻量级、1:1 原生高清、仿 QQ/Snipaste 风格的桌面截图与贴图工具。

本工具不仅支持高分辨率（DPI）感知，确保截图绝对清晰不模糊，还集成了丰富的标注工具以及实用的“钉在桌面（贴图）”功能。

---

## 🚀 核心特性

- **1:1 像素级高清**：强制开启 Windows 最高级别的 `PerMonitorV2` DPI 感知，完美支持高分屏，截图不糊、不缩放。
- **仿 QQ 精简工具栏**：支持矩形、椭圆、箭头、画笔、文本标注，以及**智能自动累加序号**（① ② ③...）工具。
- **高级贴图功能（📌 钉在桌面）**：支持将截图一键固化为置顶浮窗，支持鼠标拖动调整位置，双击或右键菜单可快速关闭或另存为。
- **完备的撤销机制**：支持对历史标注进行逐步撤销。
- **全局快捷键唤醒**：后台默默守护，随时随地一键唤起。

## 📂 项目结构与代码简介

核心代码文件（如 `main.py`）中包含以下核心模块：

* **ScreenshotOverlay**：主全屏画布，负责屏幕截取、遮罩层绘制、鼠标框选及绘图调度。
* **ScreenshotToolbar**：仿 QQ 风格的工具栏，控制各种标注模式的切换。
* **PinnedWindow**：置顶贴图组件，实现图片“钉在桌面”的独立生命周期管理。
* **HotkeySignaler**：快捷键信号发射器，确保全局热键安全触发 PyQt 线程。

## 🎮 使用说明

### 1. 快捷键操作
* **启动截图**：按下 **`Alt + A`** 唤起全屏截图界面。
* **启动录屏/相关功能**（若有配置）：按下 **`Alt + R`**。
* **退出截图**：在截图状态下随时按下 **`ESC`** 或点击工具栏的 `× 取消`。
* **快速完成**：双击框选区域，或按下 **`Enter / Return`** 键，即可快速保存到剪贴板并退出。

### 2. 贴图（📌 钉在桌面）操作
* 点击工具栏的 **`📌 钉在桌面`** 按钮，当前截图会固化在屏幕上。
* **移动贴图**：鼠标左键按住贴图，可任意拖动其在桌面的位置。
* **关闭贴图**：双击贴图任意区域，或者在贴图上点击右键 -> 选择“关闭”。
* **保存贴图**：在贴图上点击右键 -> 选择“另存为...”。
