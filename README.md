# 🚀 CopySnap - 仿 QQ 桌面独立截图工具

这是一个基于 **Python + PyQt6** 开发的独立桌面截图应用程序。它完全脱离 QQ 运行，支持全局快捷键唤醒，具备高分屏 1:1 原生高清画质，并集成了丰富的标注与贴图功能。

## ✨ 核心特性
- 🎹 **全局热键**：按下 `Alt + A` 瞬间唤醒截图。
- 🔍 **1:1 原生高清**：针对 Windows DPI 缩放优化，截屏绝对不模糊。
- 🛠️ **全功能工具栏**：支持矩形、椭圆、箭头、自由画笔、文本键入。
- 🔢 **智能序号**：支持 ① ② ③ 智能累加序号，制作教程必备。
- 📌 **📌 钉在桌面**：支持将截图变成置顶浮窗，双击即可关闭。
