一个基于 **PyQt6** 打造的轻量级、1:1 原生高清、仿 QQ/Snipaste 风格的桌面截图与贴图工具。

- **仿 QQ 精简工具栏**：支持矩形、椭圆、箭头、画笔、文本标注，以及**智能自动累加序号**（① ② ③...）工具。
- **高级贴图功能（📌 钉在桌面）**：支持将截图一键固化为置顶浮窗，支持鼠标拖动调整位置，双击或右键菜单可快速关闭或另存为。
- **完备的撤销机制**：支持对历史标注进行逐步撤销（`Ctrl + Z` 视效）。
- **全局快捷键唤醒**：后台默默守护，随时随地一键唤起。
